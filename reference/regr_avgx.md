# DuckDB function regr_avgx

Returns the average of the independent variable for non-NULL pairs in a
group, where x is the independent variable and y is the dependent
variable.

## Usage

``` r
regr_avgx(y = DOUBLE, x = DOUBLE)
```

## Arguments

- y:

  `DOUBLE`

- x:

  `DOUBLE`

## Value

`DOUBLE`
