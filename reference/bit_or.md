# DuckDB function bit_or

Returns the bitwise OR of all bits in a given expression.

## Usage

``` r
bit_or(arg)
```

## Arguments

- arg:

  `TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## Value

`TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## SQL examples

    bit_or(A)
