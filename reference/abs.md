# DuckDB function abs

Absolute value.

## Arguments

- x:

  `TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | FLOAT | DOUBLE | DECIMAL | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT`

## Value

`TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | FLOAT | DOUBLE | DECIMAL | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT`

## Overloads

- `abs(x = TINYINT)`

- `abs(x = SMALLINT)`

- `abs(x = INTEGER)`

- `abs(x = BIGINT)`

- `abs(x = HUGEINT)`

- `abs(x = FLOAT)`

- `abs(x = DOUBLE)`

- `abs(x = DECIMAL)`

- `abs(x = UTINYINT)`

- `abs(x = USMALLINT)`

- `abs(x = UINTEGER)`

- `abs(x = UBIGINT)`

- `abs(x = UHUGEINT)`

## SQL examples

    abs(-17.4)
