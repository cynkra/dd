# DuckDB function bit_xor

Returns the bitwise XOR of all bits in a given expression.

## Usage

``` r
bit_xor(arg)
```

## Arguments

- arg:

  `TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## Value

`TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## SQL examples

    bit_xor(A)
