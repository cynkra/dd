# DuckDB function sha256

Returns a `VARCHAR` with the SHA-256 hash of the `value`.

Returns a `VARCHAR` with the SHA-256 hash of the `blob`.

## Arguments

- value:

  `VARCHAR`

- blob:

  `BLOB`

## Value

`VARCHAR`

## Overloads

- `sha256(value = VARCHAR)`

- `sha256(blob = BLOB)`
