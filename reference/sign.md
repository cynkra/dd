# DuckDB function sign

Returns the sign of x as -1, 0 or 1.

## Arguments

- x:

  `TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | FLOAT | DOUBLE | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT`

## Value

`TINYINT`

## Overloads

- `sign(x = TINYINT)`

- `sign(x = SMALLINT)`

- `sign(x = INTEGER)`

- `sign(x = BIGINT)`

- `sign(x = HUGEINT)`

- `sign(x = FLOAT)`

- `sign(x = DOUBLE)`

- `sign(x = UTINYINT)`

- `sign(x = USMALLINT)`

- `sign(x = UINTEGER)`

- `sign(x = UBIGINT)`

- `sign(x = UHUGEINT)`

## SQL examples

    sign(-349)
