# DuckDB function regr_avgy

Returns the average of the dependent variable for non-NULL pairs in a
group, where x is the independent variable and y is the dependent
variable.

## Usage

``` r
regr_avgy(y = DOUBLE, x = DOUBLE)
```

## Arguments

- y:

  `DOUBLE`

- x:

  `DOUBLE`

## Value

`DOUBLE`
