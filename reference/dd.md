# DuckDB functions

A list of known DuckDB functions.

## Usage

``` r
dd
```

## Examples

``` r
dd[1:3]
#> $`!__postfix`
#> function(x = INTEGER) {
#>   stop("DuckDB function !__postfix() is not available in R.")
#> }

#> <environment: namespace:dd>
#>
```
