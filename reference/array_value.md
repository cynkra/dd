# DuckDB function array_value

Creates an `ARRAY` containing the argument values.

## Usage

``` r
array_value()
```

## Value

`ARRAY`

## SQL examples

    array_value(1.0::FLOAT, 2.0::FLOAT, 3.0::FLOAT)
