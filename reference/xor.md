# DuckDB function xor

Bitwise XOR.

## Usage

``` r
xor(left, right)
```

## Arguments

- left, right:

  `TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## Value

`TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## SQL examples

    xor(17, 5)
