# DuckDB function bit_and

Returns the bitwise AND of all bits in a given expression.

## Usage

``` r
bit_and(arg)
```

## Arguments

- arg:

  `TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## Value

`TINYINT | SMALLINT | INTEGER | BIGINT | HUGEINT | UTINYINT | USMALLINT | UINTEGER | UBIGINT | UHUGEINT | BIT`

## SQL examples

    bit_and(A)
